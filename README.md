# LinAlg Bootcamp 🚀

선형대수와 코사인 유사도를 인터랙티브하게 학습할 수 있는 웹 애플리케이션입니다.

## 🌟 주요 기능

- **인터랙티브 학습**: 벡터, 내적, 코사인 유사도를 시각적으로 이해
- **실시간 계산기**: 코사인 유사도를 직접 계산해보기
- **시각화**: 2D 벡터 그래프로 개념 파악
- **Python 코드 예제**: 실제 구현 방법 학습

## 📚 학습 내용

1. **벡터의 기초** - 벡터의 정의와 연산
2. **내적 (Dot Product)** - 벡터 내적의 개념과 계산
3. **벡터의 크기** - 벡터 크기 계산 방법
4. **코사인 유사도** - AI/ML의 핵심 유사도 측정 방법

## 🚀 GitHub Pages 배포 방법

### 1. Repository 생성
1. GitHub에서 새 repository 생성
2. Repository 이름은 원하는 대로 설정 (예: `linalg-bootcamp`)
3. Public으로 설정

### 2. 파일 업로드
1. 이 repository의 `index.html` 파일을 다운로드
2. 생성한 repository에 업로드

### 3. GitHub Pages 설정
1. Repository의 **Settings** 탭으로 이동
2. 왼쪽 메뉴에서 **Pages** 선택
3. **Source** 섹션에서:
   - **Deploy from a branch** 선택
   - **Branch**: `main` (또는 `master`) 선택
   - 폴더: `/ (root)` 선택
4. **Save** 버튼 클릭

### 4. 배포 완료
- 몇 분 후 `https://[your-username].github.io/[repository-name]/` 에서 사이트 확인 가능
- 주소는 Settings > Pages에서 확인할 수 있습니다
